Project Goal

I plan to use two public credit card application datasets to answer the following questions through the extraction, analysis, and visualization of data.

1. Which factors have the biggest impact on credit card approval?

2. Are there any discrimination based on race, income, sex or maybe other factors?

To answer the first question, I will be building a prediction model using logistic regression. To answer the second question, I will be using Oaxaca-Blinder-Kitagawa decomposition, a regression analysis I learnt in my Economics of Discriminiation class to isolate and quantify discrimination from differences in application result.

Collaboration Plan

There is no collaboration plan because Jingzhi Yang will be the only one doing this project

Datasets

The following two datasets have all the necessary information I need in order to answer the questions posted above. I also provided descriptions of two datasets as well as how I am going to use them.

Dataset 1: Real bank Credit Approval dataset

This is a public dataset from Kaggle created by user Seanny. It is from a real bank with customer names coded in ID format.

Dataset 2: Cleaned Credit approval dataset from UCI

This Kaggle dataset is originally from UCI machine learning repository. Kaggle User SAMUEL CORTINHAS have cleaned the original UCI data by filling missing values and inferring feature names from the raw dataset. This step is necessary in order to get more context and make this dataset easier to use. I will be using this cleaned version as a starting point of my analysis.

In this project I will be using both datasets at the same time because they have very different features of an applicant. I will be using the first dataset to answer the first question about most predictive factor because the first dataset has detailed financial information about applicants. I will be using the second dataset to answer the second question about discrimination because the second dataset contains detailed demographic information about applicants.
